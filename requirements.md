# 要件定義書 — 自由診療専用ページ (private-services-site)

## 1. プロジェクト概要
クリニックの提供する自由診療（自費診療：プラセンタ、高濃度ビタミンC点滴、サプリメント等）について、メニュー、料金、詳細情報を求職者や患者に向けて分かりやすく美しく提示するWebサイトを新規作成します。

---

## 2. 目的と背景
- **目的**: 自由診療メニューを一覧化し、スプレッドシートのみで金額や内容の更新、メニューの追加・停止を行える仕組みを導入し、運用コストを削減する。
- **デザイン**: 自由診療にふさわしい「プレミアムな信頼感」と「高級感」を感じられるUI/UXを提供する。

---

## 3. 実装アプローチ

### フロントエンド
- HTML5, Vanilla CSS, Vanilla JSによる静的ページ構成。
- 画面読み込み時に、裏側でGAS (Google Apps Script) の配信APIからメニュー情報を非同期でロード（fetch）。
- GASが一時的に応答しない場合のフォールバックとして、ローカルの `data/menu_fallback.json` からメニュー情報を読み込んで表示する。

### 運用・データ構造（スプレッドシート）
- **管理用スプレッドシート**: [自由診療メニュー管理スプレッドシート](https://docs.google.com/spreadsheets/d/1PxPhCTPaV_XMOF3KV968igCD0tLjvygFuz-w3ByVzlo/edit?gid=0#gid=0)
- カラム構成: `status`, `id`, `category`, `title`, `price`, `description`, `detail`, `featured`, `badge`
- `status` が `表示中` のものだけをサイトに表示。
- `featured` にチェックを入れたものは「おすすめ」マークを付与して強調表示。

### CI/CD
- GitHub Actions を使用して、`main` ブランチへの Push 時に FTP経由で自動ビルド・デプロイを行う。

---

## 4. UI/UX デザイン設計

- **カラーテーマ**: 
  - メインアクセント: シャンパンゴールド (`#d4af37`, HSL: 47, 66%, 53%)
  - 背景色: 優しく清潔感のあるオフホワイト (`#faf8f5`) およびピュアホワイト (`#ffffff`)
  - テキスト: 視認性が高く上品なダークスレートグレー (`#2c3e50`)
- **カテゴリ切り替えタブ (Tab Menu)**:
  - 画面上部に固定または滑らかにスクロールするカテゴリボタンを配置。「注射・点滴」「美容」「検査」などのカテゴリを選択すると、該当するメニューがフェードインして切り替わります。
- **価格カード (Menu Cards)**:
  - 美しいグリッドレイアウトでメニューをカード表示。
  - タイトル、価格（目立たせる）、説明、そして「詳細」を開くアコーディオン構造。
